UserProperties

  • id : String
    Unique identifier of the user.
  • email : String
    Email of the user.
  • birthDate : date
    Date of birth of the user. Used for age validation.
  • creationDate : date
    The date user was created.
  • username : String
    Username of the user.
  • address : LocationProperties
  • phoneNumber : String
    Phone number of the user.
  • password : String
    Password of the user.
