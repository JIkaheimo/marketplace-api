PostProperties

  • id : String
    Identification number of the posting
  • description : String
    Description of the posting
  • title : String
    Short title of the post.
  • category : String
    Category of the item being sold
    Enum:
    computers
    electronics
    cars
    pets
    food
    drinks
  • askingPrice : BigDecimal
    Price of the product
  • posted : Date
    Server generated datestamp when the post was created.
  • deliveryType : PostProperties_deliveryType
  • seller : SellerProperties
  • location : LocationProperties
  • imageUrls : List
    Images of the product being sold.
