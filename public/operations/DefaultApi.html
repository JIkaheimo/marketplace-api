This is the API

createPost

Creates a new post.

URL

http://localhost:3000/api/posts

HTTP Method

POST

Response Type

PostProperties

Parameters

  • Body: body PostProperties(PostProperties)

    Required data in correct format to create a new post.

deletePost

Deletes a post with the given id. To delete a post the user has to be authenticated and authorized.

URL

http://localhost:3000/api/posts/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id String

    Unique resource identifier.

  • Header: Authorization String

    User specific bearer token.

displayImage

Returns an image with the specified name/path.

URL

http://localhost:3000/api/images/{name}

HTTP Method

GET

Response Type

byte[]

Parameters

  • Path: name String

    Name or path of the image.

displayImages

Returns path of all the images.

URL

http://localhost:3000/api/images

HTTP Method

GET

Response Type

List

Parameters

displayPost

Displays a post with the given id.

URL

http://localhost:3000/api/posts/{id}

HTTP Method

GET

Response Type

PostProperties

Parameters

  • Path: id String

    Unique resource identifier.

displayPosts

Displays paged posts.

URL

http://localhost:3000/api/posts

HTTP Method

GET

Response Type

List

Parameters

  • Query: offset BigDecimal

    The number of items to skip before starting to collect the result set.

  • Query: limit BigDecimal

    The number of posts to return.

loginUser

Authenticates the user to use some Marketplace functionalities. Returns user token and username.

URL

http://localhost:3000/api/login

HTTP Method

POST

Response Type

inline_response_200_1

Parameters

  • Body: body LoginProperties(LoginProperties)

    Username and password for authentication.

modifyPost

Modifies a post with the given id.

URL

http://localhost:3000/api/posts/{id}

HTTP Method

PUT

Response Type

PostProperties

Parameters

  • Path: id String

    Unique resource identifier.

  • Body: body PostProperties(PostProperties)

    Required data in correct format to modify a post.

  • Header: Authorization String

    User specific bearer token.

registerUser

Registers a new user. Registering for the API authenticates the user to create, update and delete posts.

URL

http://localhost:3000/api/users

HTTP Method

POST

Response Type

inline_response_200

Parameters

  • Body: body UserProperties(UserProperties)

    Valid request body for registration.

searchPosts

Performs a search query for posts based on given parameters.

URL

http://localhost:3000/api/posts/search

HTTP Method

POST

Response Type

List

Parameters

  • Body: body Body(Body)

uploadPostImages

Replaces and updates post specific images with the provided ones. The user must be authenticated and authorized to updload images.

URL

http://localhost:3000/api/posts/{id}/upload

HTTP Method

POST

Response Type

Parameters

  • Path: id String

    Unique resource identifier.

  • Form: filename List(byte[])
